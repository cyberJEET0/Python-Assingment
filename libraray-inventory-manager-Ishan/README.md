# Library Inventory Manager

A simple Python-based library management system using Object-Oriented Programming, JSON persistence, and a command-line interface.

## Features
- Add, issue, return, and search books
- Persistent storage in JSON (`catalog.json`)
- Logging of operations (`library.log`)
- Modular package structure
